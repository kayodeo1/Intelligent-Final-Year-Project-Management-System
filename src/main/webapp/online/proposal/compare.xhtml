Submitted Proposal — by Adewale Kayode
Project Title
Intelligent Student Project Management System Using NLP-Based Duplicate Detection
Problem Statement
Many universities lack an automated system to manage final year projects and detect duplicate or highly similar proposals. Supervisors must manually review each submission, which is time-consuming and error-prone. This results in duplicate projects going undetected, wasting institutional resources and undermining academic integrity.
Aim & Objectives
  1. To develop a web-based project management platform for final year students
  2. To integrate NLP-based similarity detection for submitted proposals
  3. To provide supervisors with tools to review and approve projects efficiently
Methodology
The system will use the Agile development methodology. The backend will be built with Java EE (JSF/PrimeFaces), PostgreSQL with pgvector for vector storage, and LangChain4j with the Ollama embeddinggemma model for generating text embeddings. Cosine similarity via pgvector will be used to compare proposal embeddings against existing projects and other submitted proposals.
Matched Project — by Chinedu Okafor (2024)
Project Title
Automated Final Year Project Allocation and Plagiarism Detection System
Problem Statement
The manual process of allocating and managing final year projects in higher institutions is inefficient. There is no mechanism to detect when students submit project topics that are identical or closely related to previously completed ones, leading to repeated work and dilution of academic standards.
Aim & Objectives
  1. To design an automated system for managing student project allocations
  2. To implement text similarity analysis to flag duplicate project topics
  3. To evaluate the system's accuracy in detecting similar submissions
Methodology
The project followed the waterfall model. It was implemented using PHP and MySQL for the web platform. TF-IDF vectorization combined with cosine similarity was used to compare project proposals. The system was tested with a dataset of 150 past project titles and abstracts from the department's records.
