Project Title
Intelligent Project Management System with NLP-Based Similarity Detection
Problem Statement
Many universities face the recurring problem of students submitting final year project proposals that are too similar to previously completed projects. Manual checking is tedious and error-prone, leading to duplicated research efforts and wasted academic resources.
Aim & Objectives
1. To design a web-based project management system for supervising final year projects 2. To implement NLP-based similarity detection using sentence embeddings 3. To evaluate system effectiveness through comprehensive testing
Methodology
The system will adopt an agile development methodology. The backend will be built using Java EE with PrimeFaces for the UI. NLP similarity detection will leverage LangChain4j with Ollama embeddings stored in PostgreSQL using the pgvector extension. Cosine similarity will be computed between proposal embeddings to flag potentially duplicated topics.
